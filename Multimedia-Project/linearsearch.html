Linear Search


int arr[]={3,4,1,2,7,6,8,5};

int value=6,arrSize=8;

bool found=false;

for(int i=0;i<arrSize;i++){

     if(arr[i]==value){

          found=true;

          break;

     }

}

if(found) printf("Found\n");

else printf("Not found\n");


>Linear Search is finding a value in a list from the start until the end or the value is found.

>Complexity O(N)
