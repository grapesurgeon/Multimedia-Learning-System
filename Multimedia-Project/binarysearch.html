Binary Search

int arr[]={1,2,3,4,5,6,7,8};

int value=6,arrSize=8;

bool found=false;

int left=0,right=arrSize-1,mid;

while(left<=right){

     if(arr[mid]==value){

          found=true;

          break;

     }

    

     if(arr[mid]<value){

          left=mid+1;

     }

    

     else{

          right=mid-1;

     }

}

if(found) printf("Found\n");

else printf("Not found\n");


>Binary Search is finding a value in a sorted list by eliminating half on each iteration until it the range of data is     1 or the value is found.

>Complexity O(log(N))
